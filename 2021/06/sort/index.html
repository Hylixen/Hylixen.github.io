排序算法

  2021-06-10   algorithm   排序  Comments Word Count: 1754words Read Count: 4minutes

常见排序算法

分析排序算法性能
  1. 执行效率（时间复杂度）
    • 最好时间复杂度（best）
    • 最坏时间复杂度（worst）
    • 平均时间复杂度 （avg）
  2. 内存 （空间复杂度）
  3. 稳定性

O($n^2$)排序算法

冒泡排序（bubble sort）

每次遍历后将最大值交换到最右侧，遍历规模减1。（不变性）每次都会选取出最大值到右侧有序区左端，有序区保持有序，（单调性）每次遍历后遍历规模减小1，因此算法具有正确性。

效率

  • worst： $O(n^2)$
  • best: $O(n)$
  • avg: $O(\frac{n(n-1)}{4})=O(n^2)$
    排序的交换次数与逆序度相等，分析平均复杂度时依据逆序度进行分析，$满序度=\frac{n(n-1)}{2}$,$逆序度=满序度-有序度$，逆序度最小为$0$,最大为$\frac{n(n-1)}{2}$，取均值为$O(\frac{n(n-1)}{4})=O(n^2)$。

内存

$O(1)$ in place

稳定性

  • 比较时相等不交换元素稳定

代码实现

template<class T>
void bubble(vector<T>& a){
  cout<<__func__<<":"<<endl;
  int size = a.size();
  if(size<=1)
    return;
  bool f = false;
  for(int i=0;i<size;++i){
    for(int j=0;j<size-i-1;++j){
      if(a[j]>a[j+1]){
        swap(a[j],a[j+1]);
        f = true;
      }; 
    }      
    if(!f){
        break;
      };
  }
}

插入排序（insertion sort）

将数据分为有序无序左右两个区间，每次从无序右区间取左端点加入有序区间，有序区间规模加1，乱序部分规模减1。（不变性）每次都会选取出无序区间元素插入到左区间，左区间保持有序，（单调性）每次插入后无序区间规模减小1，因此算法具有正确性。插入排序与冒泡排序都是 $O(n^2)$,in place排序算法，但插入排序性能更好，应用更多，冒泡排序交换操作常数是3而插入排序的移动操作常数是1.

效率

  • worst： $O(n^2)$
  • best: $O(n)$
  • avg: $O(n^2)$
    数组插入平均复杂度为$O(n)$，插入n次，因此平均复杂度为$O(n^2)$。

内存

  • $O(1)$ in place

稳定性

  • 仅在有序元素>插入元素时移动稳定

代码实现

void insertion(vector<int>& a){
  cout<<__func__<<":"<<endl;
  int size = a.size();
  if(size<=1)
    return;
  for(int i=1;i<size;++i){
    int val = a[i];
    int j = i - 1;
    for(;j>=0;--j){
      if(val>=a[j]){
        break;
      }
      a[j+1] = a[j];
    }
    a[j+1] = val;
  }
}

选择排序 （selection sort）

类似插入排序，也分已排序区间和未排序区间。但是选择排序每次会从未排序区间中找到最小的元素，将其放到已排序区间的末尾。

效率

  • worst： $O(n^2)$
  • best: $O(n^2)$
  • avg: $O(n^2)$

内存

  • $O(1)$ in place

稳定性

  • 在swap时可能交换相等元素顺序，不稳定 5* 5 2 –> 2 5 5*

代码实现

void selection(vector<int>& a){
  cout<<__func__<<":"<<endl;
  int size = a.size();
  if(size<=1)
    return;
  for(int i=0;i<size;++i){
    for(int j=i+1;j<size;++j){
      if(a[i]>a[j])
        swap(a[i],a[j]);
    }
  };
};

O($nlog(n)$)排序算法

归并排序 （merge sort）

利用递归，每次对左右两个区间进行归并排序，直到区间只有不多于两个元素，将有序的子区间数组再自下而上依次合并。

效率

  • worst： $O(nlogn)$
  • best: $O(nlogn)$
  • avg: $O(nlogn)$
    复杂度分析:
    $$\begin{aligned}T(n) &=T(\frac{n}{2}+O(n)) \
    &= 2(2T(\frac{n}{2})+\frac{1}{2}O(n))+O(n)\
    &= 2^2T(\frac{n}{2^2})+2O(n) \
    &= 2^kT(\frac{n}{2^k})+kO(n) \
    &= nT(1)+lognO(n) \
    &= O(nlogn) \end{aligned}$$

内存

  • $O(n)$

稳定性

  • 前半段、后半段有相同值时优先合并前半段时稳定

代码实现

void merge(vector<int> &a,int lo,int hi){
  if(hi<=lo)
    return;
  int mid = (hi+lo) >> 1;
  int tmp[hi-lo+1];
  int i = lo;
  int j = mid + 1;
  int k = 0;
  while(i<=mid&&j<=hi){
    if(a[i]<=a[j]){
      tmp[k++] = a[i++];
    }
    else{
      tmp[k++] = a[j++];
    };
  };
  if(i<=mid){
    while(i<=mid)
      tmp[k++] = a[i++];
  };
  if(j<=hi){
    while(j<=hi)
      tmp[k++] = a[j++];
  };
  while(k>0){
    a[lo+k-1] = tmp[k-1];
    --k; 
  };
};

void merge_sort(vector<int> &a,int lo,int hi){
  if(hi-lo<=1){
    if(hi>lo&&a[lo]>a[hi]){
      swap(a[lo],a[hi]);
    };
    return;
  };
  int mid = (hi+lo)>>1;
  merge_sort(a,lo,mid);
  merge_sort(a,mid+1,hi);
  merge(a,lo,hi);
};
  • 归并排序复杂度为$O(nlogn)$效率稳定但不是in place限制了使用。

快速排序 （quick sort）

应用减治思想选取pivot将小于pivot元素放于于左侧，将大于pivot元素放于右边，直至区间长度为1，自上而下排序。

效率

  • worst： $O(n^2)$
  • best: $O(nlogn)$
  • avg: $O(nlogn)$
    数组有序时可能会退化成$n^2$,选取pivot时加入随机可使复杂度稳定在$O(nlogn)$。

内存

  • $O(1)$

稳定性

  • 含有swap操作不稳定

代码实现

int partition(vector<int> &v, int lo, int hi){
  int val = v[hi];
  int q = lo;
  for(int i=lo;i<hi;i++){
    if(v[i]<val){
      swap(v[q++],v[i]);
    }
  }
  swap(v[q],v[hi]);
  return q;
}

void quick_sort(vector<int> &v, int lo, int hi){
  if (lo>=hi)
    return;
  int q = partition(v,lo,hi);
  quick_sort(v,lo,q-1);
  quick_sort(v,q+1,hi);
};

O($n$)排序算法

桶排序 （bucket sort）

将数据分为m桶后排序

效率

  • $O(m\frac{n}{m}log(\frac{n}{m}))$

内存

  • 取决于内部算法，快排$O(n)$,归并$O(n+n)$

稳定性

  • 取决于内部算法，快排不稳定，归并稳定

代码实现

void bucket_sort(vector<int> &a,int n){
  int size = a.size();
  if(size<=1)
    return;
  int _min=a[0];
  int _max=_min;
  for(auto x:a){
    _min = min(x,_min);
    _max = max(x,_max);
  };
  int step = (_max-_min)/n;
  vector<vector<int>> b(n);
  for(auto x:a){
    int i = (x/step)-1;
    if(i<0){
      b[0].push_back(x);
      continue;
    }
    if(i>=n){
      b[n-1].push_back(x);
      continue;
    }
    b[i].push_back(x);
  };
  for(auto &v:b){
    quick_sort(v,0,v.size()-1);
  };
  int i = 0;
  for(int k=0;k<n;k++){
    for(int x:b[k]){
      a[i++] = x;
    };
  };
}

应用情景

  • 数据分布均匀可分区间，比较适合外部排序，大量数据在磁盘上，内存无法全部读取。

计数排序 （count sort）

通过计数和前缀和映射出元素有序位置。

效率

  • $O(n+val)$， val为数据范围

内存

  • O(val)

稳定性

  • 从后往前取数映射时稳定

代码实现

void count_sort(vector<int>& a){
  int size = a.size();
  if(size<=1)
    return;
  int _min=a[0];
  int _max=_min;
  for(auto x:a){
    _min = min(x,_min);
    _max = max(x,_max);
  };
  vector<int> c(_max-_min+1,0);
  for(auto x:a){
    c[x-_min]++;
  }
  for(int i=1;i<c.size();i++){
    c[i] += c[i-1];
  }
  vector<int> r(size,0);
  for(auto x:a){
    r[c[x-_min]-1] = x;
    --c[x-_min];
  };
  int i = 0;
  for(auto  x:r){
    a[i++] = x;
  };
}

应用情景

  • 数据范围有限且为正整数，为负数、浮点数时要转换为等价正整数后再处理。

基数排序 （radix sort）

按位由低到高排序

效率

  • $O(kO(n))$

内存

  • 取决于内部算法

稳定性

  • 取决于内部算法

应用情景

  • 数据可划分为高低位且位间有递进关系
